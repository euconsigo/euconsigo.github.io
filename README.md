# euconsigo.github.io

Questões e atividades para alunos do Ensino Médio — uma coleção de exercícios e páginas interativas para praticar e acompanhar desempenho.

## Sobre
Este repositório contém o site estático "Desafio responder questões!" com páginas para seleção de modo de participação (público, privado ou anônimo) e fluxos iniciais de login/uso. O objetivo é oferecer um ambiente simples para aplicar exercícios e registrar resultados, respeitando opções de privacidade.
